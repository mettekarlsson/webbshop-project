# Webbshop Project

Denna webbutik är fullt med funktionell e-handelsplattform som är utformad för att ge en smidig online shoppingupplevelse. Kunder kan bläddra bland produkter och lägga till varor i kundkorgen. Webbshoppen har ett användarvänligt gränsnitt, produktkategorisering och produktbeskrivning modal. 

## Installation

#### Med VS Code
1. Installera VS Code.
2. Installera en extension som heter "Live Server".
3. Klona repots URL och kör kloningskommandot i terminalen. 
``` bash
git clone https://github.com/mettekarlsson/webbshop-project.git 
```
4. Högerklicka på HTML filen och klicka på "Oppen with Live Server".
5. Datorns standardwebbläsaren automatisk öppnar förhandsvisning till HTML-sidan.

## Funktionnalitet
- Visa produkter, sortera efter kategori
- Lägg till / ta bort produkter i kundvagnen
- Uppdatera antal och totalpris
- Responsiv design
